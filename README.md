# Python Algorithms

Python算法练习项目，包含资源分配优化和布局优化算法实现。

## 项目列表

### 1. 节点资源分配 (Resource Allocation)
- **路径**: `Python节点资源分配/`
- **算法**: 遗传算法(GA)、贪心算法
- **应用场景**: 云计算资源调度、任务分配优化

### 2. 模块布局优化 (Layout Optimization)
- **路径**: `Python模块布局优化/`
- **算法**: 模拟退火、粒子群优化(PSO)
- **应用场景**: VLSI芯片布局、PCB元件摆放

## 技术栈

- **语言**: Python 3.8+
- **核心库**: NumPy, Pandas, Matplotlib
- **Jupyter**: 算法可视化与调试

## 快速开始

```bash
# 安装依赖
pip install numpy pandas matplotlib jupyter

# 运行资源分配算法
cd "Python节点资源分配"
python gara.py

# 运行布局优化算法
cd "Python模块布局优化"
python main.py
```

## 算法说明

### 遗传算法 (Genetic Algorithm)
- 种群初始化
- 选择、交叉、变异操作
- 适应度评估与迭代优化

### 模拟退火 (Simulated Annealing)
- 初始温度设置
- 邻域搜索策略
- 温度衰减与收敛判断
